# Vibe Commerce

A modern e-commerce application built with React and Express.js.

## Features

- Product listing with dynamic pricing
- Shopping cart functionality
- Quantity controls for products
- Responsive design
- Backend API for cart management

## Tech Stack

- Frontend: React + Vite
- Backend: Express.js
- Styling: CSS
